CausalPipe-Transfer
Streaming Domain Adaptation for Causal Inference Pipelines
This repository contains the code and reproducibility materials for my research paper on CausalPipe-Transfer, a framework for estimating average treatment effects (ATE) in non-stationary streaming environments across multiple domains.
---
What This Paper Is About
A core challenge in applied causal inference is that real-world data distributions change over time. A causal model trained on last month's retail transactions may give misleading estimates today if customer behavior has shifted. Standard approaches either ignore this (static models) or retrain everything from scratch when drift is detected (expensive and often unnecessary).
CausalPipe-Transfer takes a different approach: it diagnoses which component of the causal pipeline has been affected by the shift — the propensity model, the outcome model, or both — and updates only that component. This selective adaptation strategy is both computationally efficient and, in some cases, more accurate than full retraining.
The framework is validated on three domains:
Synthetic benchmarks under five types of distributional shift
FreshRetailNet-50K — a fresh food retail dataset with 335,437 transactions
PhysioNet 2019 Sepsis — 20,000 ICU patient encounters from the CinC Challenge 2019

Estimator
All experiments use the same Augmented Inverse Probability Weighting (AIPW) doubly robust estimator:
$$\hat{\tau} = \frac{1}{n} \sum_{i=1}^{n} \left[ \mu_1(X_i) - \mu_0(X_i) + \frac{T_i(Y_i - \mu_1(X_i))}{e(X_i)} - \frac{(1-T_i)(Y_i - \mu_0(X_i))}{1 - e(X_i)} \right]$$
where $e(X_i)$ is the propensity score and $\mu_t(X_i)$ are counterfactual outcome predictions.
Confidence intervals are computed from influence-function standard errors.
---
Shift Detection
The propensity-calibration shift detector uses two signals:
Covariate shift: KS test on propensity score distributions with Bonferroni correction (threshold α/p where p = number of features, α = 0.05)
Label shift: Two-sided Welch t-test on outcome residuals from the source model (α = 0.05, plus mean shift > 2σ threshold)
Detection accuracy: 95% across 40 test cases (10 seeds × 4 abrupt shift types).
---
Notes on Reproducibility
All random seeds are fixed (seed = 42 for model fitting, seeds 0–9 for experiment repetitions)
Seed 8 produces anomalous results in Mixed and Gradual shift conditions across all methods including Oracle — this reflects a challenging data configuration, not a bug. Median MAE is more representative than mean for these conditions.
Retail bootstrap uses subsamples of 50,000 from 335,437 transactions for computational feasibility
Sepsis analysis uses the first 20,000 patients from Training Set A (alphabetical file order)
